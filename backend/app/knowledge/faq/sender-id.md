# Sender ID registration

A Sender ID is the name or number recipients see as the message origin.

1. Open **Settings**, then **Sender IDs**.
2. Click **Register Sender ID** and enter the alphanumeric or numeric ID.
3. Provide the required documents for the destination country.
4. Submit for approval. Review usually takes 1-3 business days per country.

Only approved Sender IDs appear in the campaign sender dropdown. Some countries require
pre-registration with the local carriers.
