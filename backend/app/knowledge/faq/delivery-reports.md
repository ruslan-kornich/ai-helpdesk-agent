# Viewing delivery results (DLR)

Delivery reports (DLRs) tell you whether each message reached the handset.

1. Open the **Reports** section, then **Delivery**.
2. Filter by campaign, date range, or Sender ID.
3. Each row shows the status: `delivered`, `failed`, `expired`, or `pending`.
4. Export the report as CSV from the top-right **Export** button.

Statuses update in near real time as carriers return DLRs. A `failed` status includes
an error code you can match against the error reference.
